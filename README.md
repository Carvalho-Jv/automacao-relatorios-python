# Automação de Relatórios com Python

Projeto de automação de relatórios utilizando Python e Pandas.

## Objetivo
Automatizar a análise de dados de vendas e gerar relatórios consolidados.

## Tecnologias utilizadas
- Python
- Pandas

## Funcionalidades
- Cálculo de faturamento total
- Agrupamento de vendas por produto
- Geração automática de relatório
